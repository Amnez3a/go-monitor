# Go-monitor

## CLI Программа для мониторинга серверов

### Использование

*Go-monitor* берет данные о серверах с [servers.json](). Или же можно указать путь к файлу через флаг --file

```bash
./bin/go-monitor --file $USER/nameFile.jsonc 
```
